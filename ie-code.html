What is Import Export Code (IEC)?

Import Export Code (IEC) is a 10-digit unique number issued by DGFT. It is mandatory for any business involved in importing or exporting goods/services from India. No business can import or export without IEC.

  • Mandatory for all import/export businesses
  • Valid for lifetime - no renewal required
  • Required for customs clearance
  • Eligible for export benefits and subsidies
  • Required for foreign currency transactions

Note: IEC registration is free and valid for lifetime. No renewal required!

Documents Required:

  • PAN Card of Business/Individual
  • Aadhar Card
  • Business Address Proof
  • Bank Account Statement
  • Passport Size Photograph
